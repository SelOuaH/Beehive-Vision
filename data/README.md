Example images from the supplementary Varroa destructor images for the Varroa class of the public dataset, + the used images to enhance the class separability. 

These samples illustrate supplementary mite-infested honeybee frames as described in the accompanying publication “Image-Based Honeybee Colony Conditions Detection Using a Hybrid CNN–ANN Framework” (see paper for acquisition details).
